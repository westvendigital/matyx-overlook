# Overlook on 100 — Inner Page Mockups

Verified visual mockups for each inner page, styled to match the approved homepage build.
Developer handoff / build guideline. Uses **relative links**, so it works on GitHub Pages,
Netlify, or opened locally.

## Preview on GitHub Pages
1. Create a repo and upload **all files/folders in this directory** to the repo root.
2. Repo **Settings → Pages** → Source: *Deploy from a branch* → Branch: `main`, folder `/ (root)` → Save.
3. Wait ~1 min. Your site: `https://<username>.github.io/<repo>/`
   - Index hub: `https://<username>.github.io/<repo>/`
   - Each page, e.g.: `https://<username>.github.io/<repo>/amenities/`
