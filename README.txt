OVERLOOK ON 100 — INNER PAGE MOCKUPS (Netlify deploy)
=====================================================
Verified visual mockups for each inner page, styled to match the approved
homepage build. For developer handoff / build guideline.

HOW TO DEPLOY
1. Go to app.netlify.com  ->  Add new site  ->  Deploy manually.
2. Drag this ENTIRE folder onto the drop zone (or zip-upload it).
   Deploy at the site ROOT (links use root-relative /slug/ paths).
3. Each page is then live at its own URL, e.g.  https://<your-site>.netlify.app/amenities/
